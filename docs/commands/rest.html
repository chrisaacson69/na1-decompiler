Rest

Stand your daimyo down for several seasons to recuperate. Not a turn-skip — a multi-turn commitment that locks the daimyo out of every command until it ends.

What you see

The Rest animation, rendered from ROM bytes.

The resting/recuperation scene (id 32) — a short 2-frame clip. FROM ROM

Produced headless by tools/run-animation.py 32, which runs the game’s own animation scripting VM (bank 14 $80AD, id 32 from the descriptor table at $AF80) and intercepts every CHR upload, palette write and sprite placement to reconstruct each frame — real NES palette.

What it does

Rest (driver_rest $ADB3) is the only command that spends more than one turn. It is gated on your daimyo being present ($6DA2); you’re prompted Seasons and enter a count (number_input, 1–10), which is written to that daimyo’s slot in rest_turns_remaining ($6D67, a per-daimyo counter). The game replies It will do you good. and plays the recuperation animation (ui_helper_e80c(32)). While the counter is non-zero the daimyo sits out: the per-turn logic checks rest_turns_remaining in at least five places across the AI, event and combat banks and skips the daimyo’s action when it’s set. The cost is pure tempo — you trade action-slots (the binding constraint of the game) for recovery.

How it works

The driver reads as a step-by-step script:

  1. $ADC0Daimyo-present gate ($6DA2) — only a daimyo in the fief can rest; otherwise bail.
  2. $ADD8Prompt "Seasons" → number_input(1..10) → rest_turns_remaining[daimyo].
  3. $ADDF"It will do you good." + animation 32.
  4. $902BEach following turn, the AI/event/combat logic sees the non-zero counter and skips this daimyo’s action until it ticks back to zero.
The action-economy command. Everything else in the game spends one of your four seasons-per-year slots; Rest spends several at once, voluntarily. That only pays off because the daimyo is recuperating — the multi-season lock is the price of recovery. It also explains a subtle interaction: a resting daimyo can’t be made to act, so the AI-side checks (is_battleside_province_aistate5_and_not_resting, the rebellion and natural-death passes) all special-case the rest counter before doing anything to that lord.
