How it works

The driver $AA1F reads as a step-by-step script:

  1. $AA2FSelect Give and the province; prompt "(Peasnts/Men)?" (effect_give_a) picks the recipient group.
  2. $AA35Eligibility check (effect_give_c).
  3. $AA44If you have no peasants / no soldiers → "You have no peasants!" / "You have no soldiers!" and abort.
  4. $AA5CLimit check. If the target stat is already at the header ceiling → "We’re at our limit."
  5. $AA8APrompt "(Gold/Rice)?" then "how much" (effect_give_b → give_transfer_apply).
  6. $A8D3Transfer applies — the source resource drains 1:1, then the target helper(s) run.
  7. $AA93The animation plays — ui_helper_e80c(6) hands id 6 to the bank-14 animation VM.
  8. $AA98The giving daimyo gains +1 charisma ($AA98).
  9. $AAA0"Thank you, my lord." → confirm.

The formula

Read directly from $A8D3 bytecode and execution-verified.

// SOURCE drain — 1:1, no attrition   (give_transfer_apply $A8D3)
src    = (asset == Gold) ? gold : rice
amount = number_input(cap = src)
src   −= amount

// TARGET gain — Grow’s kernel, but “current” = the average of a thematic PAIR
// Peasants mode runs BOTH of:
loyalty += give( pair = (output + loyalty)÷2 )    // give_loyalty $891D
wealth  += give( pair = (town   + wealth )÷2 )    // give_wealth  $896F
// Men mode runs only:
morale  += give( pair = (men    + morale )÷2 )    // give_morale  $89C1

where give(pair):
   g     = ⌊amount × (6 − const_two) ÷ √(pair + amount)⌋   // ×5
   g     = max(g, 1);  g = min(g, header − field)
   field += 2 × g                            // NO drain — pure benefit

Best loyalty/morale ROI when the partner stat is high and the target is low (the pair + amount denominator stays small). Rice→Peasants is the staple loyalty pump — pair it with Send to convert a maxed fief’s rice surplus into frontier loyalty.

The widest command. Peasants mode raises two fields (loyalty and wealth); Men mode raises morale. Each target is averaged with its thematic complement before the curve — loyalty↔output, wealth↔town, morale↔men — which pulls a pumped stat back toward its neglected twin and blocks runaway single-stat stacking. Unlike Grow and Build there is no secondary drain: the only cost is the source resource (1:1), and the giving daimyo even gains +1 charisma. The earlier “3-field” observation was exactly right — Rice→Peasants really does bump loyalty + wealth together.
