Hire

Spend gold to recruit men into a garrison. Recruits arrive at mediocre stats and are blended into the army by a men-weighted average — so Hire dilutes quality.

How it works

The driver $A5F4 reads as a step-by-step script:

  1. $A5FCSelect Hire; validate the province.
  2. $A615Prompt "(Men/Ninja)?" — Men recruits soldiers (this page); Ninja opens the sabotage-mission menu (→ $A2D2, see Bribe).
  3. $A558(Men path, effect_hire_men $A553) army headroom = header − men.
  4. $A564Affordability: max men = f(gold, hire rate). No gold → "You have no gold."
  5. $A573Prompt "How many?" → number entry (capped).
  6. $A584gold −= amount × hire_rate (the per-man rate re-rolls each turn).
  7. $A593apply_hire_unit_stats: roll recruit morale/skill/arms, dilute into the army, men += amount.
  8. $A5AEThe animation plays — ui_helper_e80c(33) hands id 33 to the bank-14 animation VM.
  9. $A5C7"Lord, we now have N men." → confirm.

The formula

Read directly from $A553 bytecode and execution-verified.

// COST   (effect_hire_men $A553)
headroom = header − men                          // army cap
max_buy  = affordable(gold, gold_men_hire_rate, headroom)
amount   = number_input(cap = max_buy)
gold    −= amount × gold_men_hire_rate          // rate $6E11 — RE-ROLLS each turn

// RECRUIT STATS — random per batch   (apply_hire_unit_stats $8BF4)
morale_base = rng%20 + 40                    // 40 .. 59
skill_base  = rng%20 + 60                    // 60 .. 79
arms_base   = rng%10 + 50                    // 50 .. 59

// DILUTION — men-weighted average, capped at header (morale, skill, arms)
new_stat = min( header, (stat × men + base × amount) ÷ (men + amount) )
men += amount

Hire when current stats are low (recruits don’t drag much) or to refill after combat losses. Never hire into a small high-skill army — you’ll wreck its average. Watch the per-turn hire rate; it fluctuates, so some seasons recruiting is markedly cheaper.

Two commands behind one prompt. “(Men/Ninja)?” — Men recruits soldiers (this page); Ninja opens the sabotage-mission menu (Uprising / Revolt / Dams / Assassin / Arson at $A2D2, animation 12) — the same subsystem Bribe uses, which gets its own page. (The earlier docs labeled $A2D2 “effect_hire”; the bytecode shows it is the ninja path, and recruit-men actually lives in $A553.) For recruiting, the catch is dilution: fresh recruits (morale 40–59, skill 60–79, arms 50–59) are blended in by a men-weighted average, so hiring into a small elite garrison drags its skill down. That is exactly why elite armies are trained (no dilution — see train.html) and Hire is for restoring body-count or staffing a new fief.
