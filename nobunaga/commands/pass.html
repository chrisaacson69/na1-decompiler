Lord Command · No. 21

Pass

Skip the season. Spend an action slot on nothing — and the game lets you know how it feels about that.

What it does

Pass (driver_pass $B2A1) is the whole function: it prints What a waste., waits for a confirm (confirm_prompt), and returns 1 — consuming the action so the turn advances. No province field, no daimyo record, no roll. It exists because the turn structure needs an explicit “do nothing” when you’ve no good move (or want to bank gold/rice into the harvest without developing).

Under the hood — the verified bytecode
The shortest driver in the game — and a strategic statement. With only four action-slots a year, deliberately spending one on Pass is almost always wrong; the game agrees (What a waste.). It’s here for completeness — the one command whose entire decompiled body is three lines.
