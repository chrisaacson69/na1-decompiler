□ That font is the cartridge’s own. Every number and command name above is rendered in Nobunaga’s Ambition’s built-in font, extracted straight from the ROM by tools/render-text.py. The text pipeline was decoded — format_string → redraw_window → char_classify (tile = ascii − offset, piecewise per character range) — and the glyph bitmaps live at $B2BA in bank 0 (2bpp CHR uploaded to CHR-RAM by render_boot_title_screens). The renderer reuses that exact mapping, so any string can be drawn in-engine.
